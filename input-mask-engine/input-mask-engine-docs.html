InputMaskEngine Documentation

Pure JavaScript engine for two input styles:

  • Fixed masks like AAAA-9999-AA or (999) 999-9999
  • Free constrained fields like “letters and spaces only, max 40”
The engine is pure. It does not register listeners, mutate DOM on its own, or depend on any framework. Your UI layer decides when to call it.

Constructor

const engine = new InputMaskEngine({
  tokens: {
    H: {
      test: ch => /^[A-Fa-f0-9]$/.test(ch),
      transform: ch => ch.toUpperCase(),
    },
    S: {
      test: ch => /^[A-Za-z ]$/.test(ch),
      transform: ch => ch,
    }
  }
});

Built-in tokens

TokenMeaningTransform
AAlphabetic letterUppercase
aAlphabetic letterLowercase
9DigitNone
XAlphanumericNone
*Any single characterNone

compile(mask)

Compiles either a fixed mask string or a free-field config object.

Fixed mask example

const compiled = engine.compile('AAAA-9999-AA');

Free rule example

const compiled = engine.compile({
  type: 'free',
  token: 'S',
  max: 40
});

Core API

MethodPurpose
engine.apply(...)Strict single edit. Rejects if the whole input cannot be accepted.
engine.paste(...)Loose insert. Accepts valid characters and skips invalid ones.
engine.backspace(...)Delete backward.
engine.deleteForward(...)Delete forward.
engine.accepts(...)Checks whether apply() would be accepted.

Payload shape

{
  value: 'AB12',
  selectionStart: 4,
  selectionEnd: 4,
  input: '3',
  mask: compiledMaskOrFreeConfig
}

Return shape

{
  ok: true,
  value: 'AB12-3',
  selectionStart: 6,
  selectionEnd: 6
}

On rejection:

{
  ok: false,
  value: 'AB12',
  selectionStart: 4,
  selectionEnd: 4,
  reason: 'Character "x" is not allowed here'
}

Examples

1) Fixed mask

const mask = engine.compile('(999) 999-9999');

let state = { value: '', selectionStart: 0, selectionEnd: 0 };
state = engine.apply({ ...state, input: '9', mask });
state = engine.apply({ ...state, input: '8', mask });
state = engine.apply({ ...state, input: '7', mask });

console.log(state.value);
// (987)

2) Letters and spaces only, max 40

const nameField = engine.compile({
  type: 'free',
  token: 'S',
  max: 40
});

const result = engine.apply({
  value: 'Ankit',
  selectionStart: 5,
  selectionEnd: 5,
  input: ' Karan',
  mask: nameField
});

console.log(result.value);
// Ankit Karan

3) Paste while skipping junk

const nameField = engine.compile({ type: 'free', token: 'S', max: 40 });

const result = engine.paste({
  value: '',
  selectionStart: 0,
  selectionEnd: 0,
  input: 'Ankit Karan 123 !!',
  mask: nameField
});

console.log(result.value);
// Ankit Karan

Escaping literals

Use \ to escape a token symbol and treat it as a literal.

const literalA = engine.compile('AA\\A-99');
// third A is literal, not a token

Notes

  • Free fields are ideal for rules like “A-Z, a-z and space in any order, max 40”.
  • Fixed masks are ideal for rigid patterns like phone numbers and codes.
  • apply() is strict. paste() is forgiving.
  • This version does not implement optional tokens like A? or repeat syntax like {4,6}.
